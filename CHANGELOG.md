### [v1.0.0] - Initial Release

#### Added

* Multi-client chat server implementation.
* TCP/IP socket-based communication.
* Concurrent client handling using POSIX threads.
* Username-based user identification.
* Real-time message broadcasting.
* Client join and leave notifications.
* Graceful disconnection using `/quit`.
* Thread-safe client management using mutex synchronization.

#### Project Structure

* Added server implementation.
* Added client implementation.
* Added common header file.
